#Attendance Tracker Project Using Python
The program:

Loads an existing Excel workbook with structured rows and columns for dates, times, and student/class entries.

Matches the current system date and time with the schedule stored in the sheet.

Marks attendance as Present (P) or Absent (A) with proper color coding (green for present, orange for absent).

Records additional session data such as subject name, topic, sub-topic, and core concept.

Provides visual feedback in the terminal and ensures that attendance is not overwritten unnecessarily.

This project is ideal for:

Students learning Python automation, file handling, and Excel operations.

Schools or coaching centers that want to manage attendance digitally using Excel.

Individuals interested in building custom utility tools for task automation.
